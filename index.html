RF를 통한 경기순환시계 예측, 투자 모델 생성

개인 프로젝트(공모전) / 6개월 / github


과거 자료를 통해 경기순환시계의 지표를 예측하고 예측한 값을 사용하여 해당 국면을 분류했습니다. 분류된 국면을 토대로 백테스팅을 통해 국면별 최적의 자산군별 비중을 계산합니다. 미래의 국면을 예측하고 고객의 성향을 분석해 적절한 고객에게 최적의 비중을 가지는 포트폴리오를 제시합니다.

  • 경기순환시계의 경제지표들의 좌표를 크롤링
  • Random Forest model을 통해 과거 12개월 데이터로부터 1달 후의 데이터를 예측
  • K-means clustering을 통해 해당월의 국면을 분류
  • 분류를 토대로 기간을 나누고 Search algorithm을 통해 국면별 최적 비중을 계산
  • 고객데이터를 분석해 타겟을 선정하고 미래의 국면을 예측하여 최적의 포트폴리오를 제시

Unity를 사용한 Shooting Game 만들기

개인 프로젝트 / 6개월 / github


스테이지가 있고, Player와 Enemy, Boss가 등장하는 일반적인 슈팅 게임을 모작하였습니다. 게임을 기획하고 게임 로직을 Unity와 C#을 통하여 구현하였으며, 게임 내 UI도 구현하였습니다.

  • Player 구현 (Move, Fire, Boom, 다른 객체와의 충돌 등)
  • Enemy 구현 (Move, Fire, 다른 객체와의 충돌 등)
  • Boss 구현 (랜덤을 활용한 4가지 패턴)
  • GameManager 구현 (전반적인 게임 관리, UI 관리 등)
  • ObjectManager 구현 (ObjectPooling을 통한 관리)
